Разработка приложения для вывода графиков

Приложение обеспечивает вывод графика заданной функции на заданном отрезке.

Требования:

* график должен занимать практически всю клиентскую область;

* должны быть прорисованы оси координат (если центр осей – в клиентской области, или отсечки для указания значений по осям;

* при выделении части графика эластичным прямоугольником (а еще лучше окружностью) перерисовывается выделенная часть;

* при щелчке около в точке графика (около точки) выдаются соответствующие значения функции (x, f(x));

Для примера взять функцию f(x)=ax^2+bx+c с заданными a, b, c.
